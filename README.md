# DS_Spring2026

The dataset held in the file "post_falls_mineral_instances.csv" contains 4,708 mineral instance records within a 10 km radius of 47.7156°N, -117.0135°W (Post Falls, ID), filtered to Orthorhombic, Monoclinic, and Triclinic crystal systems. Data was retrieved from Mindat.org on February 16, 2026 using the OpenMindat Python library in Google Colab.
